# Inventory Management System

A web-based application to manage and track inventory efficiently. This system allows users to perform CRUD operations on products and helps in organizing inventory data in a structured way.

---

## Features
- Add new products (name, price, quantity)
- View all products in a table format
- Edit and update product details
- Delete products from inventory
- Search products easily
- Sort products by price and quantity
- Real-time date and time display
- Clean and user-friendly interface

---

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL

## ⚙️ Setup Instructions

1. Install XAMPP / WAMP  
2. Move the project folder to `htdocs`  
3. Start **Apache** and **MySQL**  
4. Open **phpMyAdmin**  
5. Create a new database (e.g., `inventory_db`)  
6. Import the `database.sql` file  
7. Open browser and go to:http://localhost/Project folder/
